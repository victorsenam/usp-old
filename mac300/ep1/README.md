# Compilar
`make main.o`

# Rodar
`./main.o <argumentos>`  
Os argumentos são:
- row : usar orientação a colunas
- lu : usar decomposição LU
- hres : esconder o resultado de Ax = b
- hdec : esconder o resultado da decomposição
- hinp : esconder o echo da entrada
